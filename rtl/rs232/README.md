# RS232 UART 模块

## 设计描述
标准 RS232 串行通信接口，支持：
- 可配置波特率
- TX/RX 全双工
- 8N1 数据格式

## 接口
| 信号 | 方向 | 位宽 | 描述 |
|------|------|------|------|
| clk | I | 1 | 系统时钟 |
| rst_n | I | 1 | 异步复位(低有效) |
| rx | I | 1 | 串行接收 |
| tx | O | 1 | 串行发送 |
| tx_data | I | 8 | 发送数据 |
| rx_data | O | 8 | 接收数据 |
| tx_valid | I | 1 | 发送有效 |
| rx_valid | O | 1 | 接收有效 |

## 木马分析关注点
- 空闲状态下的异常 TX 活动
- 未连接引脚的数据泄漏路径
- FSM 中的死态/隐态转换
