# AES 加密模块

## 设计描述
AES-128 加密核心，支持 ECB 模式。

## 接口
| 信号 | 方向 | 位宽 | 描述 |
|------|------|------|------|
| clk | I | 1 | 系统时钟 |
| rst_n | I | 1 | 异步复位 |
| key | I | 128 | 加密密钥 |
| plaintext | I | 128 | 明文输入 |
| ciphertext | O | 128 | 密文输出 |
| start | I | 1 | 启动加密 |
| done | O | 1 | 加密完成 |

## 木马分析关注点
- 密钥到非加密输出的数据路径
- S-Box 与标准值的差异
- 加密轮数 FSM 的完整性
- 旁路信号 (done 时序) 中的隐藏信息
